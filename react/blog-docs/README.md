# Nexo Blog/Docs Template

React template for blogs and documentation.

## Tech Stack
- React 19 + TypeScript
- Vite 6
- Tailwind CSS v4
- Lucide Icons

## Project Structure
```
src/
├── posts/       → Blog articles
├── docs/        → Documentation pages
├── sidebar/     → Navigation menu
├── categories/  → Post types
├── lib/         → Utilities
├── assets/      → Images
└── styles/      → Global CSS
```

## Getting Started
```bash
npm install
npm run dev
```

## RTL Support
Set `dir="rtl"` in `index.html` and use `font-family: Cairo`.
