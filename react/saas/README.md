# Nexo SaaS Template

React template for building SaaS applications.

## Tech Stack
- React 19 + TypeScript
- Vite 6
- Tailwind CSS v4
- Lucide Icons

## Project Structure
```
src/
├── components/ui/  → Reusable UI
├── hooks/          → Custom hooks
├── lib/            → Utilities
├── config/         → Constants
├── types/          → TypeScript types
├── services/       → API client
├── layouts/        → Page layouts
├── assets/         → Images
├── app/            → Pages
└── styles/         → Global CSS
```

## Getting Started
```bash
npm install
npm run dev
```

## RTL Support
Set `dir="rtl"` in `index.html` and use `font-family: Cairo`.
