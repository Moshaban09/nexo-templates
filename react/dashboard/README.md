# Nexo Dashboard Template

React template for admin dashboards.

## Tech Stack
- React 19 + TypeScript
- Vite 6
- Tailwind CSS v4
- Lucide Icons

## Project Structure
```
src/
├── widgets/     → Dashboard stat cards
├── charts/      → Data visualizations
├── sidebar/     → Navigation menu
├── pages/       → Dashboard pages
├── lib/         → Utilities
├── assets/      → Images
└── styles/      → Global CSS
```

## Getting Started
```bash
npm install
npm run dev
```

## RTL Support
Set `dir="rtl"` in `index.html` and use `font-family: Cairo`.
