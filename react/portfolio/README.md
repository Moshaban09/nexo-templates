# Nexo Portfolio Template

React template for personal portfolios.

## Tech Stack
- React 19 + TypeScript
- Vite 6
- Tailwind CSS v4
- Lucide Icons

## Project Structure
```
src/
├── projects/    → Work showcase
├── about/       → Bio & skills
├── contact/     → Contact form
├── components/  → UI components
├── lib/         → Utilities
├── assets/      → Images
└── styles/      → Global CSS
```

## Getting Started
```bash
npm install
npm run dev
```

## RTL Support
Set `dir="rtl"` in `index.html` and use `font-family: Cairo`.
