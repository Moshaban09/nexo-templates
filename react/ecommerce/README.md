# Nexo Ecommerce Template

React template for online stores.

## Tech Stack
- React 19 + TypeScript
- Vite 6
- Tailwind CSS v4
- Lucide Icons

## Project Structure
```
src/
├── products/    → Product catalog
├── cart/        → Shopping cart
├── checkout/    → Payment flow
├── categories/  → Product types
├── lib/         → Utilities
├── assets/      → Images
└── styles/      → Global CSS
```

## Getting Started
```bash
npm install
npm run dev
```

## RTL Support
Set `dir="rtl"` in `index.html` and use `font-family: Cairo`.
