# Nexo Components UI Template

React template for UI component libraries.

## Tech Stack
- React 19 + TypeScript
- Vite 6
- Tailwind CSS v4
- Lucide Icons

## Project Structure
```
src/
├── buttons/     → Button variants
├── cards/       → Card components
├── forms/       → Input & forms
├── modals/      → Dialog boxes
├── lib/         → Utilities
├── assets/      → Images
└── styles/      → Global CSS
```

## Getting Started
```bash
npm install
npm run dev
```

## RTL Support
Set `dir="rtl"` in `index.html` and use `font-family: Cairo`.
