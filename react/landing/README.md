# Nexo Landing Template

React template for beautiful landing pages.

## Tech Stack
- React 19 + TypeScript
- Vite 6
- Tailwind CSS v4
- Lucide Icons

## Project Structure
```
src/
├── sections/    → Page sections (hero, features, pricing)
├── components/  → UI components
├── lib/         → Utilities
├── config/      → Constants
├── assets/      → Images
└── styles/      → Global CSS
```

## Getting Started
```bash
npm install
npm run dev
```

## RTL Support
Set `dir="rtl"` in `index.html` and use `font-family: Cairo`.
